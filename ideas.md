# Brainstorm de Design - Site Voos Brasil-Catar

## Resposta 1: Minimalismo Corporativo Moderno
**Design Movement:** Corporativo Minimalista com toques de Modernismo
**Probabilidade:** 0.08

### Core Principles
- Clareza informacional acima de tudo: dados precisos e bem organizados
- Hierarquia visual forte com tipografia estratégica
- Espaço em branco generoso para respiração visual
- Foco em eficiência de navegação

### Color Philosophy
Paleta neutra com acentos em azul profundo (Qatar Airways brand alignment):
- Fundo: Branco puro (#FFFFFF)
- Texto principal: Cinza escuro (#1F2937)
- Acentos: Azul marinho (#003366) e azul claro (#0066CC)
- Destaque: Ouro sutil (#D4AF37) para elementos premium

### Layout Paradigm
- Grid assimétrico com coluna principal (70%) e sidebar (30%)
- Cards flutuantes com sombras suaves
- Seções com divisores diagonais sutis
- Navegação horizontal sticky no topo

### Signature Elements
1. Ícone de avião estilizado em SVG como marca visual
2. Linhas diagonais conectando seções (representando rotas)
3. Badges com informações de classe de viagem

### Interaction Philosophy
- Transições suaves em hover (0.3s)
- Filtros com feedback visual imediato
- Tooltips informativos em elementos complexos

### Animation
- Fade-in ao carregar seções
- Slide suave para expandir detalhes
- Pulse sutil em elementos interativos

### Typography System
- Display: Poppins Bold (títulos principais)
- Heading: Inter SemiBold (seções)
- Body: Inter Regular (conteúdo)
- Mono: JetBrains Mono (dados técnicos)

## Resposta 2: Design Ousado com Gradientes Dinâmicos
**Design Movement:** Neomorfismo com Gradientes Vibrantes
**Probabilidade:** 0.07

### Core Principles
- Profundidade visual através de gradientes e camadas
- Movimento constante mas não distrativo
- Cores vibrantes que refletem energia de viagem
- Experiência imersiva e moderna

### Color Philosophy
Gradientes dinâmicos inspirados em céus e horizontes:
- Primário: Gradiente azul-turquesa (#0066FF → #00D4FF)
- Secundário: Gradiente laranja-rosa (#FF6B35 → #FF1654)
- Neutro: Cinza escuro com toque azulado (#0F1419)
- Acentos: Amarelo vibrante (#FFD700)

### Layout Paradigm
- Seções com altura variável criando ritmo visual
- Cards com efeito de profundidade (sombras multi-camadas)
- Fundo com padrão geométrico sutil
- Navegação flutuante com glassmorphism

### Signature Elements
1. Padrão geométrico de hexágonos representando rotas
2. Ícones com gradientes internos
3. Números em grande escala para dados principais

### Interaction Philosophy
- Hover com mudança de gradiente
- Cliques com efeito de ripple
- Scroll paralax em seções principais

### Animation
- Entrada com scale + fade
- Números contadores animados
- Gradientes em movimento (subtle animation)

### Typography System
- Display: Space Grotesk Bold (títulos)
- Heading: Outfit SemiBold (seções)
- Body: Outfit Regular (conteúdo)
- Accent: IBM Plex Mono (dados)

## Resposta 3: Design Elegante e Sofisticado
**Design Movement:** Art Deco Moderno
**Probabilidade:** 0.06

### Core Principles
- Elegância através de proporções e simetria
- Detalhes ornamentais sutis mas impactantes
- Tipografia como elemento de design principal
- Luxo acessível (premium mas não excessivo)

### Color Philosophy
Paleta sofisticada inspirada em viagens premium:
- Primário: Azul marinho profundo (#0A1428)
- Secundário: Dourado elegante (#B8860B)
- Terciário: Creme/Off-white (#F5F1E8)
- Acentos: Verde esmeralda (#2D5016)

### Layout Paradigm
- Composição assimétrica com eixo central
- Linhas geométricas douradas conectando elementos
- Seções com proporção 16:9 (cinema)
- Tipografia em destaque como protagonista

### Signature Elements
1. Linhas douradas Art Deco em bordas de cards
2. Números em tipografia serif elegante
3. Ícones com estilo line-art refinado

### Interaction Philosophy
- Transições lentas e deliberadas (0.5s)
- Efeito de elevação em hover
- Revelação gradual de informações

### Animation
- Fade in com delay cascata
- Slide horizontal suave
- Glow sutil em elementos interativos

### Typography System
- Display: Playfair Display Bold (títulos)
- Heading: Lora SemiBold (seções)
- Body: Lora Regular (conteúdo)
- Mono: Courier Prime (dados técnicos)

---

## Design Escolhido: Minimalismo Corporativo Moderno

Optei pelo **Minimalismo Corporativo Moderno** porque:
1. Dados de voos exigem clareza máxima
2. Alinha-se com identidade visual da Qatar Airways
3. Facilita navegação e filtros intuitivos
4. Transmite profissionalismo e confiança
5. Escalável e acessível para todos os usuários

**Estilo Visual:** Corporativo, limpo, eficiente com toques de sofisticação através de tipografia estratégica e espaçamento generoso.
